# Historical Experiments

This directory preserves non-Claude historical experiment scripts from the CascadeMind development cycle. They are useful for understanding ablations and model-routing ideas, but they are not the canonical public entrypoints.

Use the root scripts and `scripts/run_camera_ready_experiments.py` for reproduction. Historical scripts may require additional provider credentials such as `GEMINI_API_KEY`, `OPENROUTER_API_KEY`, or Azure OpenAI variables, depending on the file.
